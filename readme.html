The Spirit of the Age: or Contemporary Portraits

by William Hazlitt

A faithful facsimile of the 1st/2nd edition, with a few minor adjustments, using the LaTeX documentation preparation system.

The book is currently out of print and in the public domain.

For more information, please visit the associated wikipedia page. You may also view page images from the original books in the images folder.

Build Instructions:

$ git clone https://github.com/metabind/spirit-hazlitt.git
$ cd spirit-hazlitt/
$ xelatex main.tex

latest build: spirit-hazlitt.pdf
